# Alex Ruddell Online

Welcome to my personal website! It's pretty bare-bones, but it does the trick.

Wish me luck :)
## Project Roadmap

- **Timeline**: An interactive, scrollable timeline detailing my education and employment history, with some fun interactive descriptions and photos. 
- **Gallery**: A stylish and modern gallery view of photographs I am in and/or have taken.
- **Link Tree**: A cool display of links to my Github and LinkedIn accounts, and other relevant links.

## Running the project

To run the project, follow the below instructions:
- Clone the repository
- Install node dependencies
- Run `npm run dev` at the project root root
- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
